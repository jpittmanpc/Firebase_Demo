Class: AP

AP

new AP()

An instance of AP is automatically created and attached to the embed layer as "ap". Always access AP functions through this automatically-created window.ap instance.

registerRegion(region, cb)

Register a geofence or beacon region with a URL callback.

When the user enters or exits the region a POST request is sent to the callback URL with JSON and keys for: "regionId", "regionType", "direction", "uid" and "pushToken"

Parameters:
Name Type Description
region geofenceRegion | beaconRegion

The region to be defined
cb errorOnlyCallback

reload()

requestLocationServices()

Requests location services permissions. This will prompt the user for permision, but only once. Once the user has responed to the alert they will never be prompted again.

This is done automatically when a geofence or iBeacon region is registered

requestPushNotifications()

Requests push notifcation permissions. This will prompt the user for permision, but only once. Once the user has responed to the alert they will never be prompted again.

This is done automatically on app launch unless the "Disable auto prompt for push notifications" setting is checked
